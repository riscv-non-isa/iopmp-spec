[#IOPMP_EXTENSIONS]
== Extensions

This chapter presents IOPMP extensions designed for specific use cases. The features described in this chapter are optional extensions to the baseline IOPMP specification. Implementations are not required to support these extensions. However, an implementation that claims conformance to an extension MUST adhere to the definitions specified herein. <<#EXT_REGS>> describes all relevant registers. The subsequent sections describe each IOPMP extension in detail.
